# Infomaniak: Uitgebreid Onderzoeksrapport

**Datum:** 6 februari 2026
**Opgesteld voor:** Bollenstreek Digitaal
**Doel:** Evaluatie van Infomaniak als e-mail en workspace provider

---

## Samenvatting

Infomaniak is een Zwitserse hostingprovider die uitblinkt in privacy, duurzaamheid en Europese datasoevereiniteit. Voor Bollenstreek Digitaal biedt het een solide alternatief voor Amerikaanse providers, met name voor klanten die GDPR-compliance en ethisch ondernemen belangrijk vinden.

**Belangrijkste bevindingen:**
- Zwitsers bedrijf met 100% eigen infrastructuur in Zwitserland
- B Corp gecertificeerd, CO2-neutraal sinds 2007
- Werknemers-eigendom (25% in handen van medewerkers)
- Volledig GDPR-compliant dankzij Zwitserse adequaatheid
- Professionele API met 60 requests/minuut limiet
- Partner/reseller programma met 10-35% korting
- Affiliate programma met tot 50% commissie

---

## 1. Bedrijfsprofiel

### 1.1 Geschiedenis en Achtergrond

Infomaniak is opgericht in 1994 in Genève, Zwitserland. Het bedrijf is uitgegroeid tot de grootste webhosting provider van Zwitserland met meer dan 200 medewerkers. In tegenstelling tot veel concurrenten is Infomaniak nooit overgenomen door een investeringsmaatschappij en blijft het onafhankelijk.

### 1.2 Eigendomsstructuur

Een uniek kenmerk van Infomaniak is de eigendomsstructuur:
- **25% in handen van werknemers** via een participatieprogramma
- **Geen externe investeerders** of beursnotering
- Beslissingen worden genomen op basis van lange termijn waarde, niet kwartaalcijfers

Dit zorgt voor stabiliteit en voorkomt dat het bedrijf wordt verkocht aan een Amerikaanse techgigant.

### 1.3 Duurzaamheid en Ethiek

Infomaniak onderscheidt zich sterk op het gebied van duurzaamheid:

| Aspect | Details |
|--------|---------|
| **CO2-neutraal** | Sinds 2007, ruim voor het mainstream werd |
| **B Corp certificering** | Voldoet aan strenge sociale en milieunormen |
| **Hernieuwbare energie** | 100% groene stroom voor alle datacenters |
| **Koeling** | Innovatieve koeling met buitenlucht, geen airco |
| **Hardware** | Servers worden 15+ jaar gebruikt in plaats van 3-5 jaar |
| **Compensatie** | Resterende uitstoot wordt gecompenseerd via gecertificeerde projecten |

### 1.4 Infrastructuur

Alle infrastructuur bevindt zich in Zwitserland:
- **Twee datacenters** in Genève en Winterthur
- **100% eigen hardware** — geen gehuurde capaciteit
- **Tier III+ equivalent** betrouwbaarheid
- **Geografische redundantie** tussen beide locaties

## 2. Privacy en Compliance

### 2.1 GDPR-status

Zwitserland heeft een **adequaatheidsbesluit** van de Europese Commissie, wat betekent dat gegevensoverdracht naar Zwitserland net zo veilig wordt beschouwd als binnen de EU.

**Belangrijke punten:**
- Data verlaat nooit Zwitserland
- Geen onderworpenheid aan CLOUD Act (VS)
- Geen toegang voor buitenlandse inlichtingendiensten
- Zwitserse privacywetgeving is vergelijkbaar met GDPR

### 2.2 Verwerking van gegevens

Infomaniak verwerkt geen e-mailinhoud voor advertenties of profilering. Dit in tegenstelling tot:
- **Google Workspace**: Scant e-mails voor spam, maar ook voor productverbeteringen
- **Microsoft 365**: Gebruikt data voor AI-training tenzij expliciet uitgeschakeld

### 2.3 Certificeringen

- ISO 27001 (informatiebeveiliging)
- ISO 14001 (milieubeheer)
- B Corp gecertificeerd
- Swiss Made Software label

## 4. kSuite: Workspace Alternatief

### 4.1 Overzicht

kSuite is Infomaniak's antwoord op Google Workspace en Microsoft 365. Het biedt een complete werkplek in de cloud.

### 4.2 Prijzen

| Plan | Prijs/gebruiker/maand | Inclusief |
|------|----------------------|-----------|
| **kSuite Free** | Gratis | Basis e-mail + 15 GB opslag |
| **kSuite Standard** | ~€5,75 | E-mail + kDrive + kMeet + 3 TB |
| **kSuite Pro** | ~€7,50 | Alles + geavanceerde beheer + onbeperkte opslag |

### 4.3 Componenten

**kDrive (bestandsopslag)**
- Vergelijkbaar met Google Drive/OneDrive
- Desktop sync client (Windows, Mac, Linux)
- Realtime samenwerking
- Versiebeheer
- Delen via links

**kMeet (videoconferenties)**
- Tot 100 deelnemers
- Scherm delen
- Geen account nodig voor gasten
- End-to-end versleuteling optioneel

**Docs (documentbewerking)**
- OnlyOffice-gebaseerd
- Compatibel met Microsoft Office formaten
- Realtime samenwerking
- Minder geavanceerd dan Google Docs, maar functioneel

**kChat (team communicatie)**
- Vergelijkbaar met Slack
- Geïntegreerd met andere kSuite tools
- Minder uitgebreid dan Slack, maar voldoende voor kleine teams

### 4.4 Vergelijking met Concurrenten

## 5. API en Automatisering

### 5.1 API Overzicht

Infomaniak biedt een REST API voor de meeste diensten:

**Beschikbare API's:**
- E-mail beheer
- DNS beheer
- kDrive bestandsoperaties
- Gebruikersbeheer
- Domeinbeheer

### 5.2 Technische Specificaties

| Aspect | Details |
|--------|---------|
| **Type** | REST API met JSON |
| **Authenticatie** | OAuth 2.0 of API tokens |
| **Rate limiting** | 60 requests per minuut |
| **Documentatie** | Uitgebreid, in Engels en Frans |
| **SDK's** | Officiële PHP SDK, community Python/Node |

### 5.3 Integratiemogelijkheden

**Native integraties:**
- Geen directe Zapier-integratie
- Geen Make.com connector
- Wel REST API voor custom integraties

**Workarounds:**
- **n8n** (self-hosted): Werkt via HTTP Request nodes
- **Custom webhooks**: Mogelijk voor de meeste events
- **Zapier via Webhooks by Zapier**: Handmatig te configureren

### 5.4 Beperkingen

De API is minder uitgebreid dan bij Google of Microsoft:
- Geen Calendar API voor volledige kalenderintegratie
- Beperkte e-mail API (meer beheer dan inhoud)
- Geen real-time webhooks voor alle events

**Conclusie API:** Voldoende voor basale automatisering, maar niet geschikt voor complexe workflows die diepe e-mail of kalenderintegratie vereisen.

## 8. Vergelijking met Alternatieven

### 8.1 Infomaniak vs. Proton Mail

| Aspect | Infomaniak | Proton Mail |
|--------|------------|-------------|
| **Focus** | Complete workspace | Privacy-first e-mail |
| **Encryptie** | Standaard (TLS) | End-to-end standaard |
| **Prijs** | Goedkoper | Duurder |
| **Opslag** | Meer | Minder |
| **Extra tools** | kSuite volledig | Proton Drive, Calendar |
| **API** | Beter | Beperkt |
| **Zakelijk gebruik** | Beter | Beperkter |

**Conclusie:** Infomaniak voor productiviteit, Proton voor maximale privacy.

### 8.2 Infomaniak vs. Mailbox.org

| Aspect | Infomaniak | Mailbox.org |
|--------|------------|-------------|
| **Land** | Zwitserland | Duitsland |
| **Prijs** | Vergelijkbaar | Iets goedkoper |
| **Interface** | Moderner | Functioneel |
| **Extra tools** | Uitgebreider | Basis |
| **Partner programma** | Ja | Nee |

**Conclusie:** Infomaniak biedt meer, Mailbox.org is simpler en goedkoper.

### 8.3 Infomaniak vs. Migadu

| Aspect | Infomaniak | Migadu |
|--------|------------|--------|
| **Model** | Per gebruiker | Per domein |
| **Prijs** | Voorspelbaarder | Goedkoper bij veel domeinen |
| **API** | Beter | Vergelijkbaar |
| **Interface** | Mooier | Simpeler |
| **Support** | Sneller | E-mail only |

**Conclusie:** Infomaniak voor standaard klanten, Migadu voor agencies met veel kleine domeinen.

## 10. Conclusie en Aanbeveling

### 10.1 Eindoordeel

Infomaniak is een **uitstekende keuze** voor Bollenstreek Digitaal, zowel voor eigen gebruik als voor klanten die privacy en duurzaamheid belangrijk vinden.

**Sterke punten:**
- Perfecte GDPR-compliance via Zwitserse adequaatheid
- Ethisch en duurzaam bedrijf (verkoopargument)
- Competitieve prijzen met goede marges
- Betrouwbare infrastructuur
- Partner programma met potentie

**Aandachtspunten:**
- Minder integraties dan Big Tech
- API voldoende maar niet uitblijkend
- Bekendheid opbouwen bij klanten nodig

### 10.2 Aanbeveling

**Implementeer Infomaniak als primaire e-mail/workspace oplossing voor:**
- Bollenstreek Digitaal zelf
- Privacy-bewuste klanten
- Duurzaamheid-georiënteerde organisaties
- Non-profits en stichtingen
- MKB zonder complexe integratie-eisen

**Houd Google Workspace/Microsoft 365 als alternatief voor:**
- Klanten met bestaande investeringen in die ecosystemen
- Complexe automatiseringsvereisten
- Grote organisaties met enterprise needs

### 10.3 Volgende Stappen

1. Aanmelden als Infomaniak Partner
2. Testaccount opzetten voor eigen evaluatie
3. Documentatie maken voor klantpresentaties
4. Eerste pilot-klant identificeren
5. Feedback verzamelen en evalueren na 3 maanden
